# Nixon Brasil

Portal editorial independente sobre Richard Nixon em português.

## Publicação

O site é estático e compatível com GitHub Pages. Em **Settings → Pages**, selecione **GitHub Actions** como fonte de publicação. O workflow `.github/workflows/pages.yml` faz o deploy automático a cada atualização da branch `main`.

## Estrutura

- `index.html`: portal completo
- `styles.css`: sistema visual responsivo
- `script.js`: menu móvel e pesquisa do acervo
- `favicon.svg`: identidade do navegador
- `.github/workflows/pages.yml`: publicação automática

## Situação institucional

Protótipo independente, sem vínculo atual com a Richard Nixon Foundation, Nixon Presidential Library ou NARA.
